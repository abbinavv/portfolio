# Abhinav Raj — Portfolio

Personal portfolio website built with Next.js 15, Tailwind CSS, and Framer Motion. Features page transition animations, a multilingual loading screen, and showcases my projects in iOS development, full-stack web, and machine learning.

## Tech Stack

- **Framework** — Next.js 15 (App Router)
- **Styling** — Tailwind CSS v4
- **Animations** — Framer Motion
- **Font** — Cabinet Grotesk
- **Deployment** — Vercel

## Pages

- `/` — Home with hero, about intro, marquee, profile stats, services, featured projects, work experience, and footer
- `/works` — Full project grid with EnVision AR App, Retail Store Management, and Carbon Footprint Calculator
- `/about-me` — Bio, competencies, tech stack, experience timeline, and certifications
